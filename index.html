Какой-то контент

Ранний доступ для пользователей. Нужно выбрать несколько человек для тестирования. Добавить их в группу. На основе их обратной связи сформировать бэклог. Из него выбрать критичные доработки и внести их перед выпуском приложения на всех пользователей. Некритичные ошибки перенести в бэклог со средним приоритетом. Их можно будет доделывать уже после выпуска приложения. !Важно: нужно несколько пользователей с детьми. Без них детский функционал мы нормально не протестируем.
